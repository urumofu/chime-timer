# 🔔 時報タイマー (Chime Timer) 仕様書

## 1. 概要
指定した「分（間隔）」および「秒（起点）」に合わせて、正確な時報（テッ・テッ・テッ・チーン）をブラウザ上で鳴らすWebアプリケーションです。Game OCR Toolなどのゲーム系補助ツールと併用して、ゲーム内の定期イベント等のタイミングを正確に把握するために使用します。

## 2. 構成ファイル
- **`index.html`**: タイマーUI、ロジック（JavaScript）、スタイル設定（CSS）をすべて内包するメインファイル。
- **`起動.bat`**: Windows環境での単体起動用バッチスクリプト。ローカルサーバーの起動とブラウザ表示を自動化します。

## 3. 主要機能
### 3.1. 設定項目
- **起点秒数 (0〜59秒)**: 時報のカウント基準となる秒数。（例: 0に設定すると毎分0秒ピッタリに鳴動）
- **間隔 (1〜60分)**: 何分間隔で時報を鳴らすか。
- **予告 (0〜5秒前)**: 本番の時報（チーン）の前に鳴らす「テッ」という予告音の回数と秒数。（デフォルト: 3秒前から）
- **繰り返し**: 有効にすると、鳴動後に次のインターバルへ向けて自動的に再設定・継続します。
- **音量設定 (0〜100%)**: ブラウザからの出力音量をスライダーでシームレスに調整可能。

### 3.2. 音声生成ロジック (Web Audio API)
音声ファイル（.mp3や.wav）に依存せず、JavaScriptの `Web Audio API (OscillatorNode)` を利用した純粋なシンセサイザー波形で音を生成します。
これにより、**音声ファイルのデコード処理や読み込み起因の遅延を完全に排除**し、ミリ秒単位での正確な鳴動を実現しています。
- **予告音（テッ）**: 高めの短音設定
- **本鈴（チーン）**: 余韻（リリース長）の長い澄んだベル音色

### 3.3. ブラウザのバックグラウンド遅延対策
ブラウザのタブが非アクティブになった際、省エネのため `setInterval` の実行間隔が間引かれたり遅延したりする仕様上の課題があります。
本アプリではこれに対処するため、以下のアプローチを採用しています。
1. 次に時報を鳴らすためのターゲット時刻（絶対的なUNIXタイムスタンプ）を事前に算出。
2. そのターゲット時刻から逆算し、予告音を含めた一連の音声スケジュールを前もって `setTimeout` に予約（スケジューリング）。
3. 万が一画面の表示（UI描画）がバックグラウンドで遅れても、予約された音声はシステムレベルで発火するため時間のズレが起こりにくい設計。

## 4. UI/UX デザイン
ダークモードを基調とし、サイバーかつ高級感のあるモダンなデザインを採用しています。
- **現在時刻 / デジタル表示**: `JetBrains Mono` フォントを用いた視認性の高いUI。
- **プログレスバー**: 次の時報発火までの残り時間を直感的に把握できる視覚的なカウントダウンのバー。
- **Audio ビジュアライザー**: 音が鳴るタイミング（「テッ」「チーン」）に合わせてリアルタイムでアニメーションするバー表示。
- **ログ機能**: 過去にタイマーが発動した履歴やシステムメッセージを折りたたみ式コンソールに記録。

## 5. 動作環境と配布方法
本タイマーは完全に独立したWebファイルとして設計されているため、以下の条件と手順で単独配布・利用が可能です。

### 5.1. 動作条件（必須環境）
- **ブラウザ**: 最新のGoogle Chrome, Microsoft Edge, Firefox, Safariなど（Web Audio APIをサポートしているモダンブラウザ）
- **Python環境 (任意)**: `起動.bat` を使用してローカルサーバー経由で起動する場合は、Python 3.x がインストールされている必要があります。

### 5.2. ファイルの配布方法
このアプリケーションを他者に配布する場合は、以下のファイル群をそのままZIPパック等にまとめて渡してください。
- `index.html`
- `起動.bat`

### 5.3. 起動手順
ユーザーの環境に合わせて、主に2つの起動方法を案内できます。
1. **カンタン起動 (サーバー不使用)**
   - `index.html` をダブルクリックして、直接ブラウザで開く。
   - ※ 外部リソース（画像や音声ファイル）に依存していないため、ローカルファイル（`file:///~`）としてそのまま動作します。
2. **バッチ起動 (通常)**
   - 付属の `起動.bat` をダブルクリックする。
   - コマンドプロンプトが立ち上がり、ローカルサーバー経由（`http://localhost:8765/`）で安全・確実に実行されます（※Pythonインストール済み環境のみ）。
